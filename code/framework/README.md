## Automation scope: [Todo list](http://todomvc.com/examples/react/#/)

- **Sanity Tests**
  - Run a test with an assertion
- **Smoke Tests**
  - Access homepage on browser
- **Functional Tests**
  - Add item to list
  - Mark item as done
  - Edit item text
  - Remove item from list

## Automation scope: [Amazon](https://amazon.com/)

- **Sanity Tests**
  - Run a test with an assertion
- **Smoke Tests**
  - Access homepage on browser
- **Functional Tests**
  - Search for book
  - Add book to cart
