## How to use [Docsify](https://github.com/QingWei-Li/docsify/)

### Install

```shell
npm i docsify-cli -g
```

### Setup

Setup a boilerplate folder

```shell
docsify init docs
```

### Preview

Preview your docs locally

```shell
docsify serve docs
```
